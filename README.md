# 📈 STATIFY 2.0 – Financial Analyst Chatbot

An LLM-powered Financial Analyst Chatbot built using **LangChain**, **Hugging Face**, **DuckDuckGo Search**, and **yfinance**. The chatbot provides real-time stock prices and the latest company news through an interactive conversational interface.

# 🏗️ Architecture Overview

The project is organized into three primary components:

### 1. Week1.py

The main entry point of the application.

Responsibilities:
- Initializes the Hugging Face LLM.
- Creates the LangChain Agent.
- Registers the Stock Price Tool and News Tool.
- Accepts user queries.
- Routes requests to the appropriate tool.
- Returns the final response generated by the LLM.

---

### 2. tool.py

Contains all custom LangChain tools used by the chatbot.

Implemented tools include:

- **Stock Price Tool**
  - Fetches real-time stock information using **yfinance**.

- **News Tool**
  - Retrieves the latest company news using **DuckDuckGo Search**.
  - Cleans and formats the news before returning it to the LLM.

---

### 3. schema.py

Contains all Pydantic schemas used throughout the project.

Responsibilities include:

- Input validation
- Structured tool arguments
- Output formatting
- Type safety between the LLM and external tools

# ⚙️ Environment Setup

## 1. Clone the repository

```bash
git clone https://github.com/<username>/STATIFY_2.0_Stellar_Stack.git

cd STATIFY_2.0_Stellar_Stack
```

---

## 2. Create a Virtual Environment

### macOS / Linux

```bash
python3 -m venv venv
source venv/bin/activate
```

### Windows

```bash
python -m venv venv

venv\Scripts\activate
```

---

## 3. Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 4. Configure Environment Variables

Copy the example environment file.

```bash
cp .env.example .env
```

Update the `.env` file with your credentials.

Example:

```env
HF_TOKEN=your_huggingface_token
```

---

## 5. Run the Chatbot

```bash
python Week1.py
```
